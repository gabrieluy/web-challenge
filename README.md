# Web Challenge

## Api

### Running the api

```bash
$ cd api
$ docker-compose up
```
For api specification go to http://localhost:3000/api/

### Running the tests

```bash
$ cd api
$ npm run tests
```

## App(Frontend)

### Running the app

```bash
$ cd app
$ ng serve
```
Go to http://localhost:4200
